# Rizz Pharma

Rizz Pharma is a pharmaceutical-site, frontend landing page design with react js and tailwindcss.

## Features
- **Search Functionality:** Search product by product name
- **Responsive Design:** Responsive and modern UI built with React.js.
- **Cart Functionality:** Clicking cart icon product and to cart and cart item length is show in navbar cart icon.
- **Tab and Slider:** tab and slider added for product section.
- **API:** use fake json api made by nodejs.

## Technologies Used

### Frontend:
- React.js
- React Router Dom
- Axios (for API communication)
- Tailwind CSS (for styling)
- Vanila Css

### Backend API:
- **Category:** https://pharmaceutical-site-api.vercel.app/api/v1/category
- **Product:** https://pharmaceutical-site-api.vercel.app/api/v1/product

## Installation and Setup

### Steps to Run Locally:

1. **Clone the repository:**
   ```bash
   git clone pharmaceutical-site 
   cd pharmaceutical-site 
   ```
2. **Setup:**
   ```bash
   cd pharmaceutical-site 
   npm install
   ```
   - Create a `.env` file in the frontend directory with the following environment variable:
     ```env
     VITE_API_URL=https://pharmaceutical-site-api.vercel.app/api/v1
     ```
   - Start the frontend development server:
     ```bash
     npm run dev
     ```
3. **Access the Application:**
   - Open your browser and navigate to `http://localhost:5173`.
